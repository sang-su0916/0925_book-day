## 🚨 문제해결 가이드

### ❌ "시스템 설정이 아직 완료되지 않았습니다" 오류
**원인**: HTML 파일에 웹앱 URL이 설정되지 않음
**해결**: 3-1 단계를 다시 확인하여 URL 정확히 입력

### ❌ 이메일이 오지 않아요
**해결 순서**:
1. **Gmail 스팸함 확인**
2. **Google Apps Script → 실행 기록** 확인
3. **testEmail() 함수** 실행해서 테스트
4. **Gmail 권한** 재승인

### ❌ 스프레드시트에 데이터가 저장되지 않아요
**해결 순서**:
1. **스프레드시트 접근 권한** 확인
2. **Google Apps Script → 실행 기록** 확인
3. **testDatabase() 함수** 실행해서 테스트

### ❌ 모바일에서 제출이 안 돼요
**원인**: HTTP 환경에서는 일부 기능 제한
**해결**: HTTPS 호스팅 필요 (GitHub Pages, Netlify 등)
